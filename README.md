# AI Code Review Assistant

## Problem Statement

Developed an AI-powered application that reviews Python, Java, and C++ code, identifies coding errors, suggests improvements, generates corrected code, and provides a quality score.

## Features

- AI Code Review
- Syntax Error Detection
- Logical Error Detection
- Corrected Code Generation
- Improvement Suggestions
- Code Quality Score
- File Upload
- TXT Report Download
- PDF Report Download

## Technologies Used

- Python
- Streamlit
- Groq API
- Prompt Engineering
- ReportLab

## How to Run

pip install -r requirements.txt

streamlit run app.py
